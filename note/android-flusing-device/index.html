Android刷机

环境

  1. OnePlus8Pro

  2. Kali Rolling

  3. adb

    Android Debug Bridge version 1.0.39 Version 1:8.1.0+r23-8 Installed as /usr/lib/android-sdk/platform-tools/adb

    sudo apt install android-tools-adb

  4. fastboot

    fastboot version 1:8.1.0+r23-8 Installed as /usr/lib/android-sdk/platform-tools/fastboot

    sudo apt install android-tools-fastboot

基础

  1. 解锁OEM

    https://source.android.com/setup/build/running

  2. 进入FastbootMode

  3. 进入RecoveryMode

  4. 安装Magisk/MagiskManager

    官方网站：　https://magiskmanager.com/

  2. 其中在刷Magisk比较有用的是boot.img及recovery.img，可以使用Magisk对boot.img或recovery.img进行修改，修改为magisk_patch.img然后对相应分区flash

    1$ adb push boot.img /storage/emulated/0/Download/
2# 手机操作`MagiskManager`修改`img`文件
3$ adb pull /storage/emulated/0/Download/magisk_patch.img
4$ adb reboot-fastboot
5# 手机重启至`FastbootMode`
6$ sudo fastboot flash boot magisk_patch.img
7$ sudo fastboot continue # sudo fastboot reboot # 也可
